Weather Data API

URL format: http://127.0.0.1:5000/api/v1/station number/date

URL example: http://127.0.0.1:5000/api/v1/10/19881025
URL format for all data of one station: http://127.0.0.1:5000/api/v1/station number/
URL format for annual data of one station is : http://127.0.0.1:5000/api/v1/yearly/(station number)/(year)/
