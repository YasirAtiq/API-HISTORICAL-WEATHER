Weather Data API

URL for data for a particular station and date format: http://127.0.0.1:5000/api/exact_date/(station_number)/(date)
URL example: http://127.0.0.1:5000/api/exact_date/10/19881025
URL format for all data of one station: http://127.0.0.1:5000/api/all_data_for_station/(station_number)
URL example: http://127.0.0.1:5000/api/all_data_for_station/10
URL format for annual data of one station is : http://127.0.0.1:5000/api/all_data_for_station_annual/(station_number)/(year)
URL example: http://127.0.0.1:5000/api/all_data_for_station_annual/10/1980
